# RADAR Data Pipeline Setup

## How it works

`scripts/generate-data.js` pulls from your APIs every 4 hours (Mon-Fri business hours PT) via GitHub Actions. It writes `data.json` which RADAR reads on each scan.

## Files to add to your repo

```
your-repo/
├── radar.html          # (already deployed)
├── data.json           # (generated by the script)
├── scripts/
│   └── generate-data.js
└── .github/
    └── workflows/
        └── refresh-data.yml
```

## GitHub Secrets needed

Go to **repo → Settings → Secrets → Actions** and add:

| Secret | Source | Required |
|--------|--------|----------|
| `ANTHROPIC_API_KEY` | [console.anthropic.com](https://console.anthropic.com) | Yes |
| `SLACK_BOT_TOKEN` | Slack app → OAuth → Bot Token (xoxb-...) | Yes |
| `LINEAR_API_KEY` | Linear → Settings → API → Personal API keys | Yes |
| `AMPLITUDE_API_KEY` | Amplitude → Settings → Projects → API Key | Yes |
| `AMPLITUDE_SECRET` | Amplitude → Settings → Projects → Secret Key | Yes |

## Slack Bot Setup

Your Slack bot needs these scopes:
- `channels:history` — read public channel messages
- `channels:read` — list channels

Install to workspace, add bot to #product channel.

## Manual trigger

You can trigger a refresh anytime:
1. Go to **Actions** tab in your repo
2. Click **Refresh RADAR Data**
3. Click **Run workflow**

## Cost

Each refresh: ~$0.01 (Haiku for context synthesis)
4x/day × 20 workdays = ~$0.80/month
