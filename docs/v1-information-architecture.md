# V1 Information Architecture

## Primary site (`noahlucas.com`)

## Goal

Establish Noah as a high-agency VP Product + entrepreneur with clear proof of work and active thinking.

## Page map

- `/` Home
- `/about` Positioning + biography
- `/notes` Notes index
- `/notes/<slug>` Note detail
- `/work` Selected initiatives and outcomes
- `/contact` High-intent contact and links

## Navigation

- Home
- Work
- Notes
- About
- Contact

## Home sections

- Hero: current positioning and value proposition
- Credibility rail: logos, roles, impact counters
- Featured work: 3 selected cases
- Latest notes: 3 recent posts
- CTA: connect on LinkedIn / contact

## About sections

- Positioning statement
- Career timeline highlights
- Operating principles
- Current focus areas

## Work sections

- Case cards with problem, action, result
- Optional deep-dive page per case

## Notes taxonomy

- `product-strategy`
- `ai-systems`
- `org-design`
- `field-notes`
- `building-in-public`

## Contact

- Lightweight form
- Direct external links (LinkedIn, optional calendar)

## Secondary objective support

- Prominent cross-link to private lab entry point (`lab.noahlucas.com`)
- Framed as "AI-native operating system" (private)
